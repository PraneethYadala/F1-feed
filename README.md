# F1FeederApp

## Getting Started

All you need to do is to clone this repository,


```
git clone https://github.com/PraneethYadala/F1-feed/
cd F1-feed
```

Then, install the dependencies:

```
npm install
```

Then, run the Application:

```
npm start
```

You can access your app at 

```
http://localhost:8000/app/
```
